Overview
Strong + Beautiful Me is a blog and newsletter focused on nutrition and beauty. I surveyed about 40 target users to help direct the brand and features on the website. The brand is a mix of delicate and dominant, using pastel colors and organic shapes with straightforward sans-serif type. I designed the site on WordPress because the client had a premium template they wanted to use. To make up for template constraints, I altered code and added plugins to adjust some features.

Tools
Hand sketching, Photoshop, Illustrator, InDesign, WordPress, CSS, PHP
